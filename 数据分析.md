# 数据分析

##### SSH 协议和远程访问

SSH 协议是一种安全的远程访问协议，可以用于远程访问服务器。主要用途为：

1. 用户加密实现远程登陆
2. 服务器之间的免密登陆

采用 RSA 算法实现**非对称加密**，需要 2 个密钥：公钥和私钥。公钥用于加密，私钥用于解密。

# 命令

- pwd 查看当前工作目录

- ls 查看当前目录下的文件和目录
  ls -a 查看当前目录下的文件和目录，包括隐藏文件
  ls -l 查看当前目录下的文件和目录，包括详细信息

- cd 切换工作目录
  ~ 表示为 home 目录
  .表示目前所在的目录
  ..表示目前目录位置的上一层目录

- mkdir 创建目录
  mkdir -p 创建目录，如果目录不存在，则创建目录

- touch 创建文件

- rm 删除文件
  rm -f 强制删除文件
  rm -rf 强制删除目录
  rm -r 删除目录

- cp 复制文件
  cp -r 复制目录
  cp -R 递归复制目录

- mv 移动文件
  mv -r 移动目录

- cat 查看文件内容

- more 查看文件内容

  - 适合大文件查看，空格键下一页，b 键上一页，q 键退出

- less 查看文件内容

- head 查看文件的开头部分内容

  - -n 指定行数

- tail 查看文件的结尾部分内容

  - -n 指定行数
  - -f 实现显示文件动态追加的内容

- echo 输出内容到文件

  - echo "内容" > 文件名 将内容输出到文件
  - echo "内容" >> 文件名 将内容追加到文件

- grep 查找文件内容,通常配合管道对命令执行的结果进行搜索

  - -i 忽略大小写
  - -n 显示行号
  - -r 递归查找
  - -l 只显示文件名
  - -c 只显示匹配的行数
  - -v 只显示不匹配的行
  - -w 只显示匹配的单词

- ps 查看进程

  - ps -ef 查看所有进程
  - ps -ef | grep 进程名 查找进程
  - ps -ef | grep 进程名 | grep -v grep 查找进程，不显示 grep 进程

- && 和 ||

  - && 表示前一个命令执行成功后，才执行后一个命令
  - || 表示前一个命令执行失败后，才执行后一个命令

- find 查找文件

  - find 路径 -选项 参数
  - -name 按名称查找
  - -type 按类型查找
  - -size 按大小查找

- which,会在环境变量$path 设置的目录里查找符合条件的文件。用于快速定位已安装并配置环境变量的软件目录

- ln: 为某一个文件在另外一个位置建立一个同步的链接，可以将链接视为文件的别名。分为硬链接和软链接
- 硬链接： 一个文件可以有多个名称，软链接的方式则是产生一个特殊的文件，该文件的内容是指向另一个文件的位置
